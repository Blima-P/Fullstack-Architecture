Repositorio de estudos de Engenharia de Software organizado como um diario tecnico de evolucao continua.  
O objetivo e consolidar fundamentos, praticar logica diariamente e manter ritmo de preparacao para desafios de codigo (LeetCode e Live Code).

## Tech Stack

| Linguagens |
| --- |
| Java, JavaScript, TypeScript, Python, Go, C#, C++, Rust, SQL, Bash, HTML, CSS |

## Como navegar neste repositorio

1. Escolha uma linguagem na raiz do projeto.
2. Dentro da linguagem, siga o fluxo de estudo pelas pastas:
   - `logica-e-algoritmos`
   - `estrutura-de-dados`
   - `arquitetura-e-padroes`
   - `desafios-pessoais`
   - `leet-code`
3. Use `live-coding` para projetos praticos executados em tempo real.
